## Installation

### Carthage

Add the following line to your `Cartfile`:

```ruby
github "nighthawk/ASPolygonKit"
```


### [CocoaPods](http://cocoapods.org)

Add the following line to your `Podfile`:

```ruby
pod "ASPolygonKit"
```
